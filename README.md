# TRIP Tool — Node Criticality Dashboard

**Transportation Resilience Identification Planning (TRIP) Tool**  
Southeastern NC Coastal Plain · Fayetteville to Wilmington

This dashboard visualizes road network node criticality scores for the 
southeastern North Carolina coastal plain. Nodes are scored based on 
proximity to essential facilities including hospitals, fire stations, 
law enforcement, gas stations, government buildings, and critical 
infrastructure.
